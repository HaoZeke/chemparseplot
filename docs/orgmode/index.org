* Welcome to chemparseplot Documentation

chemparseplot is a pure-Python parsing and plotting library for computational chemistry outputs.

* Quick Start

** Installation

#+begin_src bash
pip install chemparseplot
# With plotting support
pip install "chemparseplot[plot]"
# With ORCA NEB support (ORCA 6.1+)
pip install orca-pi
#+end_src

** Basic Usage

#+begin_src python
from chemparseplot import parse, units

# Parse ORCA output
data = parse.orca.read_log("output.log")

# Use unit-aware quantities
energy = units.Q_(1.5, "eV")
#+end_src
